- ##### `Binary-Exponentiation`
```cpp
    // find a ^ b or a**b in just O( log(b) ) time 
    ll power(ll a, ll b) {
        ll result = 1;
        while (b > 0) {
            if (b % 2 == 1)
                result = (result * a) % MOD;

            a = (a * a) % MOD;
            b /= 2;
        }
        return result;
    }
```
